## Compression

Online tools for reducing image file size without noticeable quality loss.

### General Image Compression

General-purpose online image compression tools supporting common formats like JPG, PNG, and WebP.

- [TinyPNG](https://tinypng.com) – Smart lossy compression for PNG and JPEG images.
- [Squoosh](https://squoosh.app) – Advanced image compression tool by Google with fine-grained controls.
- [ImageOptim](https://imageoptim.com/online) – Online version of the popular image compression tool.
- [ILoveIMG Compress](https://www.iloveimg.com/compress-image) – Easy-to-use image compression with batch support.
- [Optimizilla](https://imagecompressor.com) – Interactive image compression with live quality preview.
- [Compress JPEG](https://compressjpeg.com) – Simple bulk JPEG image compression online.
- [JPEG Optimizer](http://jpeg-optimizer.com/) – Classic JPEG compression tool with size and quality control.

---

### Exact File Size Compression

Tools designed to compress images to a specific target size, useful for strict upload limits.

- [Compress 50 KB](https://compress50kb.com/) – Compress images to exactly 50 KB, ideal for strict size limits.
- [Reduce Images](https://www.reduceimages.com/) – Compress images by setting a target file size.
- [Online Image Compressor](https://online-image-compressor.com/) – Compress images to a desired size in KB.

---

### Batch Image Compression

Tools optimized for compressing multiple images at once.

- [Bulk Image Compressor](https://bulkimagecompressor.com/) – Upload and compress multiple images simultaneously.
- [ILoveIMG Batch Compress](https://www.iloveimg.com/compress-image) – Batch compression with drag-and-drop support.
- [CompressNow](https://compressnow.com/) – Batch image compression with adjustable quality settings.

---

### Format-Specific Compression

Tools focused on specific image formats or modern web formats.

- [SVGOMG](https://jakearchibald.github.io/svgomg/) – Optimize and compress SVG files with detailed controls.
- [EZGIF](https://ezgif.com/optimize) – Compress and optimize GIF images online.
- [CloudConvert Image Compressor](https://cloudconvert.com/compress-image) – Compress images with format conversion support.
- [Kraken.io](https://kraken.io/web-interface) – Professional-grade image compression with API support.

---

### Privacy-Friendly / Client-Side Compression

Tools that process images locally in the browser without uploading to a server.

- [Squoosh](https://squoosh.app) – Client-side compression powered by WebAssembly.
- [SVGOMG](https://jakearchibald.github.io/svgomg/) – Local SVG optimization in the browser.
- [ImageToolbox](https://imagestoolbox.com/) – Browser-based image compression and conversion.
